    Project

  • Optical motion capture in reach to grasp movement using python

    • Optical motion capture in reach to grasp movement is a project about sensing hand movements. We will stick markers on the fingers and wrists, then we bring the movement data to plot as a graph to see abnormalities. This project is made for patients with hand movement disorders. So the patient can do the test by themselve at home and assess the initial abnormalities. As a result, the shape of the graph and the position of the fingers are very close to the actual test machine.

    • Parking management system is a project about checking parking spaces. This project is made for replace the sensors in the parking lot. Usually a sensor is used per a parking slot.
